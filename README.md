# LECRec

LECRec is a desktop app for recording lectures, generating notes and summaries, and asking questions about the content all in one place.

## Features

* **Lecture Recording** — Record audio directly in the app with Play, Pause, and Stop controls.
* **Automatic Transcription** — Recorded lectures are transcribed to text and saved as `.md` files.
* **Summarization** — Generate a summary of the current lecture with a single click.
* **Ask a Question** — A built-in chatbox lets you ask questions about the lecture content and get contextual answers.
* **Export to PDF** — Save any lecture's summary as a formatted PDF for easy sharing or printing.
* **Save & Manage Recordings** — Save, edit, clear, or delete individual lecture files.
* **Two Views**
   * **RecView** — Live recording interface.
   * **SaveView** — Review saved transcripts, notes, and past Q&A.

## Getting Started

1. Open LECRec.
2. Click Play to start recording a lecture (or select an existing file from the sidebar).
3. Use Pause / Stop to control the recording.
4. Click Save to store the transcript, or Summarize to generate a summary.
5. Click Export PDF to save the current lecture's summary as a PDF.
6. Switch to SaveView to review saved lectures.
7. Use the Ask a Question box to query the current lecture's content, then click Submit.

## File Management

* Recordings are listed in the left sidebar (e.g. `rec1.md`).
* Each saved lecture includes:
   * Transcript
   * Summary / key points
   * Bibliography (if applicable)
   * Q&A history
   * PDF export (if generated)
