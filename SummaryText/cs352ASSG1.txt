## prob1: isFactorial

Write a program, in a file `isFactorial.c` contained in the directory `prob1`.

* **Program behavior:** The program reads in a sequence of 0 or more positive integer values from stdin until no more can be read in; and for each value N so read, prints out "N = b!" if there an integer b so that N = b!, and "N not factorial" if no such b.
* **Input format:** The input is a sequence of positive integers.
* **Error Conditions:** It is an error if the input contains any non-positive integers or any non-integer values. Non-positive values in the input should not produce any output (but you should produce an error message on stderr). If the input contains something that cannot be read as an integer, your program should exit.
* **Assumptions:** You do not need to worry about the size of the integers input.

## prob2: getPrime

Write a program, in a file `getPrime.c` in the directory `prob2`.

* **Behavior:** The program reads in a positive integer from stdin and prints out the smallest prime number that is larger than n.
* **Input:** A positive integer value.
* **Output:** The smallest prime number that is larger than n, printed out using the statement `printf(“%d\n”, num);`.
* **Error Conditions:** It is an error if your program cannot read an integer from stdin or if the value read is not positive.

## prob3: evenSquares

Write a program, in a file `evenSquares.c` in the directory `prob3`.

* **Behavior:** The program reads in an integer n from stdin and prints out all the even squares that are less than or equal to n.
* **Input:** A positive integer value n that is greater than 3 and less than 2,000,000,000.
* **Output:** The even perfect squares that are less than or equal to n, one per line. `printf(“%d\n”, num);` for each perfect square being printed out.
* **Error Conditions:** It is an error if your program cannot read an integer from stdin or if the value read is not in the correct range.
* **Assumptions:** You do not need to worry about someone entering an number larger than the largest int (2,147,483,647).
